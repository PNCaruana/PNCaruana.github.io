RECENT WORK

Parametric Saliency Maps

The focus of my masters thesis was a framework for incorporating distracted pedestrians into crowd simulations in Unity (C#) and later was ported to Unreal Engine 4. The Predictive Avoidance Model (PAM) is used as a steering algorithm with distracted behaviour layered on top using behaviour trees.

Methodologies used: Unreal Engine 4 (blueprints and C++) and Unity (C#), software prototyping, usability testing, experiment piloting.

Below is a video showcasing a dynamic octree implementation to speed up neighbour search, which is a bottleneck in our algorithm. It is implemented almost entirely in C++. In order to work around Unreal's poor C++ documentation, I frequently consulted and searched through engine code. It supports 3D neighbour search. An octree could also be used to allow pathfinding in 3D. Note that performance is impacted by the recording software as well as the age of my laptop, which is several years old at the time of recording. On this machine, the octree implementation led to a 25% increase in performance compared to our previous sphere trace approach. On a modern machine, we can simulate up to 2000 agents before performance drops. Further optimization is still in progress.

Visual Attention For Virtual Pedestrians

A probabilistic visual attention system for virtual agents. Attention distribution over the visual field is encoded into a 2D texture, which can then be sampled at runtime and used to drive agent choices. The textures are generated by convolving two gaussians, one in the horizontal direction of the visual field, and one across distance from the viewer.

Methodologies used: Unity (C#), software prototyping, usability testing, experiment piloting

Parametric Saliency Maps for Virtual Agents

Visual attention and saliency system for virtual agents implemented as a shader (HLSL) in Unity, using the visual attention textures from the previous project to modulate saliency across the visual field.

Methodologies used: Unity (C#), software prototyping, usability testing, experiment piloting
